Chatsy Widget Test

This page tests the CDN build of the Chatsy widget. A chat button should appear in the bottom-right corner.

Programmatic Usage

// NPM import
import Chatsy from 'chatsy';
const chat = new Chatsy('YOUR_AGENT_ID', {
  settings: {
    button: { backgroundColor: '#0066FF', position: 'bottom-right' },
  },
});

chat.on('message', (msg) => console.log('New message:', msg));
chat.open();

Script Tag Usage

<script src="https://cdn.jsdelivr.net/npm/chatsy@latest/dist/chatsy.min.js"
  data-agent-id="YOUR_AGENT_ID"
  data-button-background-color="#0066FF">
</script>
